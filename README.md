# CasaFind — Real Estate Listing Platform (Capstone - Lab 5)

## Project title & theme
**CasaFind** — Real Estate Listing Platform (static UI)

## Short description of sections implemented
- Header & navigation (semantic `<header>` and `<nav>`)
- Hero banner with search form and animated heading
- Property listings grid (cards) implemented with CSS Grid
- Sidebar filters (visual-only, inside `<aside>`)
- Agent cards section
- Contact form with accessible labels
- Footer with social links

## Features
- Semantic HTML5 structure: `<header>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- Responsive layout using Flexbox (header/footer) and CSS Grid (cards)
- CSS variables, hover effects, transitions, and one keyframe animation (`fadeInUp`)
- Accessible elements: `aria-label`, `role="search"`, `alt` on images, form labels, "Skip to main content"
- Images use placeholder service (`picsum.photos`) — replace with local images in `images/` folder if required

## Breakpoints used
- Desktop: >= 1024px (default layout with sidebar + 2-column grid)
- Tablet: ~768–900px (stacked layout, grid reduced)
- Mobile: <= 600px (single-column cards, stacked header)
